# 项目介绍
本项目（目前开发进度0.01%）是基于[egg.js][egg]框架开发的专门为前端提供RESTful接口的后端项目。

写本项目的初衷就是想在工作生活的闲暇之余多学点东西，后端选择node也是想将本人弱掉渣的js能力提高一点，也为将来的全栈之路打上一点微薄的基础。

现在与之配合的前端实现是正在写的一个微信小程序（也纯为练手），日后计划再利用 [vue][vue] + [element-ui][element-ui] 写个后台管理系统。这一套具体啥时候能写完就不得而知了。

核心内容就不讲了（主要没啥可讲的，都是官方文档走的），反正也没人看。

```bash

```
